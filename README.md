# [Alef Test Task](https://boisterous-vacherin-a730d6.netlify.app/)
## Stack: Vue 3 (Composion API), TS, TailwindCSS

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
